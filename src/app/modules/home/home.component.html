Grid layout is a generalized layout system with emphasis on rows and columns. There are powerfully flexible methods for defining repeating patterns of grid lines, attaching elements to those grid lines, and more.

The first step to creating a grid is defining a grid container. The child elements of a grid container become grid items. The children of those child elements do not become grid elemenets. It's possible to nest grids inside grids.

There are two kinds of grids: regular and inline grids. These are created with special values for the display property: grid and inline-grid. The first generates a block-level box, and the second an inline-level box.


          display: grid;
This grid box is placed in the middle
of a sentence.

          display: inline-grid;
This grid box is also placed in the middle
of a sentence.

"The most fundamental unit is the grid line. By defining the placement of one or more grid lines, you implicity create the rest of the grid's components."

"A grid track is a continuous run between two adjacent grid lines. In other words, a grid column or a grid row."

"A grid cell is any space bounded by four grid lines. This is the smallest unit of area in grid layout. Grid cells cannot be directly addressed with CSS grid properties."

"A grid area is any rectangular area bounded by four grid lines, and made up of one or more grid cells. Grid areas are directly addressable by CSS grid properties, which allow you to define the areas and then associate grid items with them."

You can define the grid lines in your overall grid template, or what the CSS specification calls the explicit grid. Grid lines can be referred to by number, and can also be named by the author.

* Fixed-Width Grid Tracks *

Grid tracks are a fixed width. In this context, "fixed-width" means the grid lines are placed such that the distance between them does not change of content within the grid tracks.


          display: grid;
          grid-template-columns: [start-col-a] 200px [col-b] 50% [col-c] 100px [stop end last];
          grid-template-rows: [start masterhead] 3em [content] minmax(3em, 100%) [footer] 2em [stop end];
