Grid Layout

Grid layout is a generalized layout system with emphasis on rows and columns. There are powerfully flexible methods for defining repeating patterns of grid lines, attaching elements to those grid lines, and more.

The first step to creating a grid is defining a grid container. The child elements of a grid container become grid items. The children of those child elements do not become grid elemenets. It's possible to nest grids inside grids.

There are two kinds of grids: regular and inline grids. These are created with special values for the display property: grid and inline-grid. The first generates a block-level box, and the second an inline-level box.


        display: grid;
This grid box is placed in the middle
of a sentence.

        display: inline-grid;
This grid box is also placed in the middle
of a sentence.
